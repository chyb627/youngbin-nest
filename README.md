# youngbin-nest

node_nest_backend

### install step

- npm i -g @nestjs/cli
- 설치가 완료되면 nest라는 명령어를 사용할 수 있게 된다.

- nest new ./
- 현재 폴더로 nest 초기세팅을 한다.

### module 생성

- nest g module cats(모듈명, 복수로 짓는게 일반적임)

### controller 생성

- nest g controller cats(컨트롤러명)

### service 생성

- nest g service cats(서비스명)

### middleware 생성

- nest g middleware logger
